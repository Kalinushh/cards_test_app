# Cards Test App

Приложение для работы с карточками блюд: загрузка данных, отображение, создание собственных карточек, редактирование, удаление, лайки и фильтрация. Реализовано на React, TypeScript и Redux Toolkit.

## Функциональность

- загрузка списка блюд из публичного API
- отображение карточек в сетке
- детальная страница блюда
- лайк/дизлайк и фильтрация по избранному
- создание собственной карточки
- редактирование существующей карточки
- удаление карточек
- локальное хранение данных в Redux
- деплой на GitHub Pages

---

## Маршруты приложения

Приложение использует следующие маршруты:

### `/products`
Главная страница со всеми карточками.  
Данные загружаются из API или из локального Redux-хранилища.  
Есть фильтр «Избранное».

### `/products/:id`
Детальная страница конкретного блюда.  
Отображение полной информации: изображение, категория, страна, рецепт.  
Есть возможность лайкнуть, удалить или перейти к редактированию.

### `/create-product`
Страница создания новой карточки.  
Форма с валидацией и загрузкой изображения.  
После отправки создаёт новую карточку в списке.

### `/products/:id/edit`
Страница редактирования карточки.  
Форма автоматически подставляет данные выбранного блюда.  
После сохранения обновляет существующую карточку.
